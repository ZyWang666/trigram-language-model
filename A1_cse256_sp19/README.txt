This program support three models: unigram, trigram, and trigram with laplace
smoothing. To run the program, simply run 'python data.py', and then there should
be a prompt asking which model to run. To run unigram, enter 'unigram', to run
trigram, enter 'trigram', and to run trigram with laplace smoothing, enter 
'trigram with smoothing'.
To run the adaptation model, the command is 'python adaptation.py'.
